8-Puzzle Solver with A* Algorithm

    The program uses an informed path search algorithm using different variations of the A* algorithm to determine the path of least cost, such as least distance traveled, from starting point to goal. The A* algorithm is the fastest search algorithm for any kind of given heuristic as it must expand a minimum of A* nodes. The different types of variations which can solve the eight-puzzle are uniform cost search, A* with the Misplaced Tile heuristic, and A* with Euclidean Distance heuristic.
    The program will display the expanding state with its nodes and show the values of g(n)=total cost sum from starting node to current node, and h(n)=cost from current node to goal state, for each state the algorithm finds. When the goal state is found then the program will display “Goal !!!” and state how many nodes were expanded as well as the maximum number of nodes in the queue at a time before terminating. This project is written using C++.
